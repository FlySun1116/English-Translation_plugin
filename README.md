# Sun Tech English Helper

> 技术英语记忆控制塔 - 智能查词与学习辅助工具


## ✨ 功能特性

### 🎯 快速查词
- **快捷键支持**：选中单词后按 `Alt + Q` 即可查看翻译
- **智能翻译**：自动调用 Google Translate API 获取中文翻译
- **实时反馈**：浮窗显示翻译结果，4秒后自动隐藏
- **查阅记录**：自动记录每次查阅次数和时间

### 📊 学习统计
- **单词追踪**：记录每个单词的查阅次数、最后查阅时间和来源URL
- **每日统计**：按日期统计每日查询量
- **趋势分析**：可视化展示10日学习趋势
- **顽固词汇**：识别和展示查阅次数最多的单词

### 🎨 智能高亮
- **一键高亮**：快速高亮页面中所有已记录的单词
- **次数标记**：高亮标记显示查阅次数
- **智能过滤**：自动过滤脚本、样式、输入框等不需要高亮的元素
- **轻松清除**：一键清除所有高亮效果

### 💾 数据管理
- **本地存储**：所有数据存储在本地浏览器中，保护隐私
- **双视图模式**：今日查询视图和历史总榜视图
- **快速删除**：支持删除单个单词记录
- **悬停查看**：鼠标悬停单词即可查看翻译
